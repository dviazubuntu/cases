Section 02
Scope of Work

Scope is delivered in five phases, matching the roadmap defined in the BeeGuard 4.0 Technical Architecture & Deployment Specification. Each phase has a fixed deliverable set and an explicit exit gate.

PhaseIn-scope deliverables
P0 — DiscoveryApiary site survey, hive format validation, connectivity/power assessment, data-sovereignty legal review, UAE bee species identification, requirements sign-off.
P1 — POC5–10 hive pilot. Operator-assisted capture kit (wearable + stand-mount camera). Edge inference deployment. Basic CV pipeline (bee count, colony strength, brood pattern). Simple dashboard. UAE-species model fine-tuning begins.
P3 — ProductionProduction deployment across ADAFSA apiary sites (50-hive production-ready environment). Three-tier KPI & Metrics Framework live: all 19 metrics across Operator, Manager, and ADAFSA Strategic tiers. Queen and Varroa detection. Predictive Honey Yield Forecast, Bloom Readiness Index, Colony Collapse Early Warning, Total Hive Health Index. LLM daily briefings (EN/AR). Mobile inspection app. UAE PASS SSO. Regulatory compliance export. 99.5% uptime SLA established.
P4 — OptimiseMonthly ML retraining cycles, accuracy improvement tracking, additional detection classes, Beekeeper Network data aggregation. Modelled as an initial 3-month retainer, renewable thereafter.

Basis of estimate: Scope, phase gates, and architecture referenced in this proposal are unchanged from the BeeGuard 4.0 Technical Architecture & Deployment Specification (v3), including the operator-assisted frame inspection model confirmed with ADAFSA.

Section 09
Assumptions & Exclusions
Assumptions
  • Phase durations use the midpoint of each range stated in the BeeGuard 4.0 Technical Architecture & Deployment Specification (v3).
  • The 19-metric, three-tier KPI & Metrics Framework (BeeGuard 4.0 Technical Overview v3, Section 9) is delivered by the Analytics Engine and Dashboard API roles already costed in Phases 3–4 below; it does not introduce additional staffing or budget beyond the figures in Section 4.
  • All hourly rates are taken directly from the SoftTeco Rate Card (AED) with a uniform +10% uplift applied per client instruction. No role outside the published rate card has been used.
  • A standard 40-hour work week is used to convert FTE allocation into billable hours.
  • Phase 4 (Optimise) is costed as an initial 3-month retainer; actual ongoing retainer terms to be confirmed at P3 gate. Phases P0 and P1 reflect a 20% cost reduction relative to the original staffing plan, applied via proportional reduction of hours at unchanged hourly rates.
  • Hardware unit pricing is indicative and subject to confirmation via vendor RFQ at Phase 0.
  • Production-ready hardware (Section 6) is based on a 50-hive environment (AED 212,600). Additional wearable and stand-mounted camera units may be purchased by ADAFSA at any time as new hives are connected; no platform changes are required for hardware expansion.
  • UAE PASS SSO integration assumes ADAFSA provides API access and sandbox credentials during Phase 2.
  • ADG Cloud / G42 Core42 account provisioning and any associated platform fees are assumed to be arranged directly by ADAFSA and are excluded from this estimate.
Exclusions
  • Cloud infrastructure consumption costs (compute, storage, bandwidth) — billed directly by the UAE sovereign cloud provider, not included in labour or hardware figures above.
  • Third-party LLM API consumption costs (if external Claude API mode is selected over the local LLM fallback) — billed directly by Anthropic per ADAFSA's own account.
  • Mobile carrier SIM/data plans for field connectivity (4G LTE private APN).
  • Physical site preparation, electrical work, or civil works at apiary locations.
  • Travel and accommodation for on-site visits beyond what is included in standard remote delivery — to be agreed separately if on-site presence is required.
  • Annual hardware maintenance, replacement parts, and extended warranty beyond original equipment manufacturer terms.
  • VAT and any applicable UAE taxes — all figures in this proposal are presented net of VAT.
