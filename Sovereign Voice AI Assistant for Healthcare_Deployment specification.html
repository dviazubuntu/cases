NOTE

The language model is hosted directly on the Blackwell accelerator, whose 96 GB of VRAM comfortably holds the model in a single card, while each concurrent agent's speech pipeline runs on a dedicated L40S. This separates the shared, memory-heavy LLM tier from the parallel, latency-sensitive voice tier — letting agent count scale by adding L40S cards without touching the model tier.

Production-grade GPUs

For a production sovereign healthcare deployment, the data-center-class L40S and RTX PRO 6000 Blackwell are preferred over the consumer GeForce RTX 5090: ECC memory, passive rack cooling, vGPU support, and a licensing model that permits data-center use.

MIG partitioning

The RTX PRO 6000 Blackwell supports Multi-Instance GPU — up to four fully isolated instances per card — which complements the hybrid model where strict resource isolation between tenants or agents is required.

A single SR675 V3 populated with 1 × RTX PRO 6000 Blackwell (LLM tier) plus up to 4 × L40S (one per agent) hosts the entire recommended hybrid topology — Configuration B — and all three containers in one 3U chassis, with the dedicated data node alongside in the same rack.

Why a tier-1 OEM

For a sovereign per-hospital deployment, an NVIDIA-certified platform from a major OEM (Lenovo, Dell, HPE) brings ECC, redundant power, validated thermals, and regional service contracts — preferable to a self-assembled build in a clinical environment.

Indicative cost

Configuration- and GPU-dependent, quote-based through the OEM channel. A fully populated single-server deployment typically lands in the tens of thousands USD — sized separately from the software license.
