Voicemail System Architecture

IMS-based voicemail with the Client's TTS integration

Integration Overview

  • Protocol: SIP (Session Initiation Protocol) between Operator IMS Core and your VMS
  • SIP Commands: INVITE, ACK, BYE, REGISTER, OPTIONS, CANCEL, REFER
  • Media Server: Custom GoLang-based implementation for SIP handling and media processing
  • TTS Request: POST to Client API for greeting synthesis (WAV/MP3 response)
  • AI Processing: POST recorded voicemail to Client REST API with audio file and metadata
  • Results Delivery: Client sends ASR transcription, LLM analysis, and sentiment via webhook to your endpoint
  • Response Time: TTS 200-500ms, AI processing async (webhook callback within seconds)

1. SIP Signaling

Operator IMS Core sends SIP commands (INVITE, ACK, BYE, etc.) to establish call session with your GoLang media server

2. Greeting Playback

GoLang media server checks Redis cache or requests TTS synthesis from Client. Generated greeting plays to caller before recording begins

3. Message Recording

System records voicemail and stores in PostgreSQL/MinIO. Audio file and metadata sent to Client REST API for processing

4. AI Processing

Client performs ASR transcription, LLM analysis, and sentiment detection. Results delivered to your webhook endpoint asynchronously

Caching Strategy

Phrase Type Example Redis TTL Hit Rate
Static templates "Leave a message after the beep" 7 days ~80%
Personalized "You have reached +375291234567" 24 hours ~15%
Dynamic "Today is April 7th, time is 2:35 PM" Not cached ~5%

Key Benefits

  • Clear separation: Client handles TTS + AI processing, you control SIP and call flow
  • Async processing: Webhook-based AI results don't block call handling
  • Low latency TTS: 95% of greeting requests complete in <500ms with caching
  • Complete AI suite: ASR transcription, LLM analysis, and sentiment detection in one service
  • Scalable: Client handles compute-intensive AI workloads, your VMS stays lightweight
  • Flexible billing: Pay per TTS request and per voicemail processed
